Ce fichier est un placeholder pour le dossier Assets.

Les fichiers suivants sont normalement requis pour une application WinUI 3:
- LockScreenLogo.scale-200.png
- SplashScreen.scale-200.png
- Square150x150Logo.scale-200.png
- Square44x44Logo.scale-200.png
- Square44x44Logo.targetsize-24_altform-unplated.png
- StoreLogo.png
- Wide310x150Logo.png

Pour tester l'application, vous pouvez:
1. Utiliser des images de 200x200px pour les logos carrés
2. Utiliser une image de 620x300px pour le logo large
3. Ou télécharger les assets par défaut de Microsoft

L'application fonctionnera même sans ces fichiers, mais aura des avertissements lors de la compilation.
